# 🎨 LINE Animated Sticker Creator

LINEアニメーションスタンプをブラウザだけで作成できるWebアプリです。  
画像のアップロードから背景透過、フレーム分割、プレビュー、APNG書き出しまで、すべてブラウザ内で完結します。

> **プライバシー安全** — 画像データはサーバーに送信されません。すべてブラウザ内で処理されます。

---

## ✨ 主な機能

| 機能 | 説明 |
|------|------|
| 📤 画像アップロード | PNG / JPG / GIF / WebP に対応。複数選択・ドラッグ&ドロップ可 |
| 🎨 背景透過 | カラーピッカー＋スポイトで背景色を選択、許容誤差スライダーで微調整 |
| 🔲 フレーム分割 | スプライトシートのグリッド分割 or 個別画像をフレームとして使用 |
| ▶ プレビュー | リアルタイムアニメーションプレビュー。再生速度・ループ回数を調整可能 |
| 💾 APNG書き出し | LINE仕様に準拠したAPNGファイルを生成＆ダウンロード。ZIPエクスポートも対応 |
| 📋 仕様チェック | LINEスタンプの仕様（サイズ・フレーム数・ファイルサイズ等）を自動チェック |

## 🚀 使い方

### 必要なもの
- モダンブラウザ（Chrome / Edge / Firefox / Safari）
- インターネット接続（初回のライブラリ読み込みに必要）

### ローカルで起動

```bash
# リポジトリをクローン
git clone https://github.com/miraichi-ai/line-sticker-creator.git
cd line-sticker-creator

# ローカルサーバーを起動（Python 3の場合）
python3 -m http.server 8080

# ブラウザで開く
open http://localhost:8080
```

> 他のサーバーでもOKです（`npx serve`、VS CodeのLive Serverなど）

### ステップバイステップ

#### ① 画像をアップロード
- フレーム画像（個別の連番PNG）をまとめてアップロード
- または1枚のスプライトシート画像をアップロード

#### ② 背景を透過（スキップ可能）
- **すでに背景透過済みの画像** → 「スキップして次へ」をクリック
- **背景を透過したい場合** →
  1. 「🎯 スポイト」ボタンを押して、画像の背景部分をクリック
  2. 「許容誤差」スライダーで色の範囲を調整
  3. 「エッジのなめらかさ」で境界を調整
  4. 「✨ 背景を透過する」をクリック
  5. 必要に応じてブラシツールで手動修正

#### ③ フレーム分割
- **個別画像モード**: アップロードした画像をそのまま各フレームとして使用
- **グリッド分割モード**: スプライトシートを行×列で分割

#### ④ プレビュー・調整
- アニメーションをリアルタイムでプレビュー
- 再生時間（1〜4秒）やループ回数を調整
- LINE仕様チェックで問題がないか確認

#### ⑤ エクスポート
- 「🎬 APNG を生成」でファイルを生成
- ファイル名を入力して「💾 ダウンロード」で保存
- ZIPでまとめてダウンロードも可能

## 📏 LINE スタンプ仕様

| 項目 | 仕様 |
|------|------|
| 画像サイズ | 最大 320×270 px（幅または高さが270px以上） |
| フレーム数 | 5〜20フレーム |
| 再生時間 | 1〜4秒（整数秒） |
| ループ | 1〜4回（再生時間×ループ ≤ 4秒） |
| ファイルサイズ | 300KB 以下 |
| ファイル形式 | APNG |

### ⚠️ 【重要】LINE審査エラーを防ぐ「安全なフレーム数」

LINEのシステムは非常に厳密で、**「1フレームあたりの秒数（ミリ秒）が割り切れない」場合に審査エラー**となります。
（例：14フレームで3秒にすると、1フレーム214.285...ミリ秒となり、浮動小数点計算の誤差で「再生時間は1, 2, 3, 4秒のいずれかに設定してください」というエラーが出ます）

エラーを防ぐため、以下の**「割り切れる」フレーム数**を使用してください。
当アプリの仕様チェック機能では、割り切れない組み合わせに対して警告を出します。

- **1秒の場合**: 5, 8, 10, 20フレーム
- **2秒の場合**: 5, 8, 10, 16, 20フレーム
- **3秒の場合**: 5, 6, 8, 10, 12, 15, 20フレーム
- **4秒の場合**: 5, 8, 10, 16, 20フレーム

## 🛠 技術スタック

- **フロントエンド**: HTML / CSS / JavaScript（フレームワーク不使用）
- **APNG生成**: [UPNG.js](https://github.com/nickerals/UPNG.js) + [pako](https://github.com/nickerals/pako)
- **ZIP生成**: [JSZip](https://stuk.github.io/jszip/)
- **背景除去（オプション）**: [@imgly/background-removal](https://github.com/nickerals/@imgly/background-removal)
